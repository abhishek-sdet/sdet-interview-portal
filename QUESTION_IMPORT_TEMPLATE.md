# Sample Questions Template for Bulk Import

Use this format in your Word document (.docx) to bulk import questions:

---

Q: What is the purpose of software testing?
A) To find bugs
B) To write code
C) To design UI
D) To deploy applications
Correct: A

Q: Which of the following is a testing framework?
A) React
B) Selenium
C) Node.js
D) MySQL
Correct: B

Q: What does API stand for?
A) Application Programming Interface
B) Advanced Program Integration
C) Automated Process Interaction
D) Application Process Interface
Correct: A

Q: What is regression testing?
A) Testing new features
B) Re-testing after bug fixes
C) Performance testing
D) Security testing
Correct: B

Q: Which tool is used for API testing?
A) Photoshop
B) Postman
C) Excel
D) PowerPoint
Correct: B

---

## Format Rules:

1. Start each question with "Q:" followed by the question text
2. Use A), B), C), D) for the four options (or A. B. C. D.)
3. Specify the correct answer with "Correct:" followed by the letter (A, B, C, or D)
4. Leave a blank line between questions (optional but recommended)
5. All questions will be assigned to the criteria and difficulty level you select during import

## Tips:

- Make sure each question has exactly 4 options
- The correct answer letter must match one of the options (A, B, C, or D)
- You can also use "Answer:" or "Ans:" instead of "Correct:"
- Questions can have any length, but keep them clear and concise
